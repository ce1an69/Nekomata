# 神秘猫咪占卜师

Nekomata 的塔罗占卜由六只神秘猫咪主持。所有塔罗牌面均融入猫咪元素。

## 猫咪角色

| 名字   | 品种           | 性别 | 特征                     |
| ------ | -------------- | ---- | ------------------------ |
| 狗肉   | 暹罗猫         | 女   | 性格呆萌                 |
| 皮胖子 | 乳白色扁脸英短 | 男   | 体型偏胖                 |
| 金龙鱼 | 中长毛白猫     | 男   | 毛发蓬松                 |
| 煤球   | 黑脸暹罗猫     | 男   | 脸部深色区域比狗肉大很多 |

| 韦恩   | 奶牛猫         | 男   | 黑白双色                 |
